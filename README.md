## _CRUD made in TypeScript, Sqlite, Express, PrismaORM_

## Installation

```sh
git clone https://github.com/CodeByAllan/UserCrudApp.git .
cd UserCrudApp/
npm install
```

## Configuration

Follow the .env.example to assemble your .env

```sh
npx prisma generate
npx prisma migrate dev --name init
```

## Build

```sh
npm run build
```

## Start

```sh
npm run start
```
